LEARN HTML Table

In this article you will learn about HTML Tables

Tag Name Explanation
table Table used to represent a two-dimensioal table made of rows and columns
thead Table Header defines the headings of table columns encapsulated in table rows
tbody Table Body it contains the table data other than table heading and table footer
tfoot Table Footer uses table rows to give footer content or to summrize content at the end of a table
tr Table Row used to add rows to a table before adding table data and table headings
td Table Data can be nested inside a table row to add a cell of data to a table
